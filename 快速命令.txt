🚀 MilesXWalkerStudio - 常用命令速查

═══════════════════════════════════════════════
📋 最常用的代码命令：
═══════════════════════════════════════════════

1️⃣ 格式化代码：
   npm run format

2️⃣ 一键修复所有问题：
   npm run fix-all  

3️⃣ 检查所有问题：
   npm run check-all

═══════════════════════════════════════════════
🔧 项目运行：
═══════════════════════════════════════════════

启动开发：npm run dev
构建项目：npm run build

═══════════════════════════════════════════════
📤 Git 推送（常用）：
═══════════════════════════════════════════════

快速推送：git add . && git commit -m "更新" && git push origin main
查看状态：git status
拉取代码：git pull origin main

═══════════════════════════════════════════════
🚀 Vercel 部署：
═══════════════════════════════════════════════

自动部署：git push origin main（推送后自动部署）
手动部署：npm run vercel:deploy
查看状态：vercel ls

═══════════════════════════════════════════════
🎯 完整发布流程：
═══════════════════════════════════════════════

1. npm run fix-all
2. git add . && git commit -m "功能描述" && git push origin main
3. 等待Vercel自动部署完成
